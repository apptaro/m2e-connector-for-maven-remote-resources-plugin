This is an eclipse update site for an m2e-connector for the maven-remote-resources-plugin.

Usage Instructions
==================
1. In your Eclipse Workspace, Navigate to Help > Install New Software
2. Paste the URL http://coderplus.com/m2e-update-sites/maven-remote-resources-plugin/ into the Work with textbox, click Add
3. Once added, you can install the M2E connector by following the instructions on screen
